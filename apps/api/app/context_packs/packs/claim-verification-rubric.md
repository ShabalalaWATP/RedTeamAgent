# Claim Verification Rubric

Use this pack to fact-check specialist outputs.

## Review Rubric

- Classify each claim as source-backed fact, inference, assumption, recommendation or unknown.
- Check whether the cited source actually supports the claim.
- Check whether the claim overstates certainty, scope, compliance or risk.
- Check whether current external facts need verification before publication.
- Check whether severity and confidence match evidence quality.
- Require explicit uncertainty where evidence is partial, stale or indirect.

## Output Requirements

- Reject fabricated citations, unsupported quotes and hidden assumptions.
- Preserve minority or dissenting agent views when they affect the decision.
