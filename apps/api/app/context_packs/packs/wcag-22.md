# WCAG 2.2 Accessibility Review Lens

Use this pack for accessibility and inclusive design review.

## Review Rubric

- Perceivable: check text alternatives, captions, contrast, adaptable layout and visual clarity.
- Operable: check keyboard access, focus order, target size, timing and motion safety.
- Understandable: check labels, instructions, error messages and predictable navigation.
- Robust: check semantic HTML, assistive-technology compatibility and valid relationships.
- Include mobile and responsive states, not only desktop screens.
- Check that status messages and validation errors are exposed to assistive technologies.

## Output Requirements

- Reference concrete user impact and remediation.
- Do not claim full WCAG conformance without a dedicated accessibility audit.
