# Report Quality Gate

Use this pack for final assurance before report publication.

## Review Rubric

- Every report must contain at least one structured finding or a clear no-finding rationale.
- Source-backed findings need source locators and short evidence excerpts.
- Unsupported claims must be labelled as inference, assumption or unknown.
- Recommendations must connect to a finding, owner, next action or decision point.
- Severity must reflect impact, likelihood, exploitability and confidence.
- Reports must not claim professional sign-off, exhaustive coverage or autonomous exploitation.

## Output Requirements

- Fail closed when source locators, evidence classifications or recommendations are missing.
- Record failed checks with actionable repair notes.
