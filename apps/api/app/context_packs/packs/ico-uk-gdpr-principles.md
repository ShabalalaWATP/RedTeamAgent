# ICO UK GDPR Principles

Use this pack when a review involves personal data, user accounts, analytics, monitoring, admin tooling or data retention.

## Review Rubric

- Lawfulness, fairness and transparency: identify the lawful basis and what the user is told.
- Purpose limitation: check that data is used only for specified, legitimate purposes.
- Data minimisation: challenge collection of data that is not necessary for the stated purpose.
- Accuracy: check correction, review and stale-data handling where decisions depend on personal data.
- Storage limitation: require retention periods and deletion paths.
- Integrity and confidentiality: check access controls, encryption, logging, backup and breach handling.
- Accountability: require records of decisions, ownership and evidence that controls are operated.

## Output Requirements

- Separate legal risk, privacy risk and security risk.
- Do not provide legal advice. Flag where a qualified data protection review is needed.
